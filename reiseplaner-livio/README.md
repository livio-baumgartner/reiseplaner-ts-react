# Reiseplaner
Der Reiseplaner ist eine einfache Webanwendung.
Der Benutzer gibt Reiseziel, Dauer und weitere Infos ein.
Durch das wird automatisch eine Packliste erstellt.
Die Liste enthält Gegenstände für die Reise.
Jeder Eintrag kann mit einer Checkbox abgehakt werden.
Erledigte Dinge werden farblich markiert.
Die Liste wird übersichtlich dargestellt und bleibt strukturiert.
Ziel: Das Packen einfacher und schneller zu machen.
